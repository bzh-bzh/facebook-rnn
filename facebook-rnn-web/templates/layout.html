What is this?

I fed every single one of my Facebook private messages into a machine learning model called a neural network. From my messages, the neural network learned what my writing style was like, and generated new, simulated Facebook messages, which are displayed above.

How does it work?

I downloaded all of my personal data from Facebook(you can do that from the settings page), and wrote a tiny Python script to parse the HTML file that contained all of my personal messages, into a list of all my messages, separated by newlines.

I then fed that into torch-rnn to train recurrent neural networks until I was satisfied with the output I was getting from them. The one I finally settled on had 512 nodes per layer, and 3 layers. On my pretty old Nvidia GTX 650 card, it took around 6 hours to train.

I've got a Google Compute Engine VM running a Python script that continually runs the command to get output from my neural network. It then separates the output by newlines into messages, censors my friends' names, and uploads each message, with metadata about creation date and length, to Google Datastore, a NoSQL database.

This website is built on the Python micro-framework Flask(bit of a trend here, haha), and uses AJAX to fetch the messages you see above from the server, which is hosted on Google App Engine.

Why did you do this?

I recently read the excellent article “The Unreasonable Effectiveness of Recurrent Neural Networks”, and thought it would be fun and interesting to see what kind of output an RNN would make, given all of my Facebook messages. The output I got really surprised me! The neural network uses emoticons, makes up URLs with (mostly) valid syntax, and generally emulates the writing style I use when writing Facebook messages.

As for why I decided to host this: I've always wanted a personal website, and I thought this would be a nice first project to showcase on it! Hopefully, there'll be many more under this domain in the future :)
